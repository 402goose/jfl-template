## Status Key

| Status | Meaning |
|--------|---------|
| COLD | No contact yet |
| ENGAGED | Interacting with content |
| DM_SENT | Reached out, awaiting response |
| WARM | Responding, interested |
| HOT | Ready to act/convert |
| CONVERTED | Done what we wanted |
| DEAD | Not responding, moved on |
